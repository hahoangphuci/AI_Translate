Cài đặt Sử dụng Web FAQ

1. Đăng nhập

  1. Mở trang chủ → nhấn Đăng nhập
  2. Chọn Google hoặc đăng ký email/mật khẩu
  3. Tài khoản mới nhận gói dùng thử (token miễn phí)

2. Dashboard

Sau đăng nhập, vào Dashboard:

TabChức năng
Dịch văn bảnNhập text, chọn ngôn ngữ, dịch tức thì (≤ 5000 ký tự)
Dịch tài liệuUpload PDF hoặc Word (.docx) (≤ 50MB)
Dịch ảnhUpload hoặc Ctrl+V → OCR → dịch
Cài đặtChọn API: Google / DeepL / Gemini
Thanh toánNạp token qua QR chuyển khoản

3. Dịch văn bản

  1. Chọn ngôn ngữ nguồn (hoặc Tự động) và ngôn ngữ đích
  2. Nhập nội dung → Dịch
  3. Sao chép hoặc tải kết quả; dùng nút hoán đổi để đảo chiều dịch

4. Dịch tài liệu

  1. Kéo thả hoặc chọn file PDF/DOCX/…
  2. Chọn ngôn ngữ đích; tuỳ chọn OCR ảnh nhúng, chế độ song ngữ
  3. Nhấn Dịch → chờ tiến trình → Tải xuống

Hệ thống cố giữ màu chữ, in đậm, in nghiêng, bảng và bố cục. PDF phức tạp xử lý qua pipeline PDF→DOCX→PDF.

5. Dịch ảnh (OCR)

  1. Tab dịch ảnh → upload hoặc dán ảnh (Ctrl+V)
  2. OCR trích xuất chữ → chọn ngôn ngữ đích → Dịch

6. Token & thanh toán

  • Token bị trừ mỗi lần dịch (theo ký tự/nội dung)
  • Xem số dư trên Dashboard hoặc Profile
  • Nâng cấp Pro / Pro Max: tab Thanh toán → quét QR → chuyển khoản đúng nội dung CK
  • Hệ thống tự xác nhận qua SePay (vài phút)

7. Lịch sử

Menu Lịch sử — xem lại, tải lại hoặc xóa bản dịch cũ.

8. Hỗ trợ

  • FAQ · Liên hệ
  • Email: support@aitranslator.vn
  • Xóa tài khoản: Chính sách xóa dữ liệu

Tài liệu đầy đủ: docs/USER_GUIDE.md · Hướng dẫn cài đặt

1. Sign in

  1. Open the home page → click Sign in
  2. Use Google or email/password registration
  3. New accounts receive a free trial token package

2. Dashboard

After login, go to Dashboard:

TabFunction
TextPaste text, pick languages, instant translate (≤ 5000 chars)
DocumentUpload PDF or Word (.docx) (≤ 50MB)
ImageUpload or Ctrl+V → OCR → translate
SettingsChoose API: Google / DeepL / Gemini
PaymentTop up tokens via bank transfer QR

3. Text translation

  1. Select source (or Auto) and target language
  2. Enter content → Translate
  3. Copy or download; use swap button to reverse direction

4. Document translation

  1. Drag & drop or select PDF/DOCX/…
  2. Choose target language; optional embedded OCR, bilingual mode
  3. Click Translate → wait → Download

The system preserves colors, bold, italic, tables and layout. Complex PDFs use PDF→DOCX→PDF pipeline.

5. Image translation (OCR)

  1. Image tab → upload or paste (Ctrl+V)
  2. OCR extracts text → select target language → Translate

6. Tokens & payment

  • Tokens are deducted per translation
  • Check balance on Dashboard or Profile
  • Upgrade Pro / Pro Max: Payment tab → scan QR → transfer with exact description
  • SePay auto-confirms within minutes

7. History

History menu — view, re-download or delete past jobs.

  • Email: support@aitranslator.vn
  • Account deletion: Data deletion policy
