Installation Guide · Developer setup

Cài đặt Sử dụng Web FAQ

1. Yêu cầu hệ thống

Thành phầnGhi chú
Python 3.10+Backend Flask
MySQL / MariaDBXAMPP hoặc standalone
Tesseract OCRBắt buộc nếu dùng dịch ảnh / PDF scan
Word hoặc LibreOfficeTùy chọn — xuất PDF từ DOCX

2. Clone & cài dependencies

git clone https://github.com/duyvo26/ai-translation-system.git
cd ai-translation-system/api_base
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

3. Database

Tạo database ai_translation (utf8mb4). Chi tiết: file docs/DATABASE_SETUP.md.

cd api_base
python app/models/base_db.py --check
python app/models/base_db.py

4. Cấu hình .env

cd api_base
copy .env.example .env

Chỉnh các biến quan trọng:

  • DATABASE_URL — kết nối MySQL
  • GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET — đăng nhập Google
  • OPENAI_API_KEY / DEEPL_API_KEY / GEMINI_API_KEY — dịch AI
  • TESSERACT_CMD, OCR_LANGS_DEFAULT=eng+vie — OCR
  • SEPAY_API_KEY, PAYMENT_BANK_* — thanh toán (xem docs/SEPAY_INTEGRATION.md)

5. Tesseract (Windows)

  1. Tải Tesseract OCR và cài gói tiếng Anh + Việt
  2. Set TESSERACT_CMD=C:\\Program Files\\Tesseract-OCR\\tesseract.exe

6. Chạy server

cd api_base
python run_api.py

Mặc định: http://127.0.0.1:5055 (frontend + API cùng port).

7. Docker (tùy chọn)

docker-compose up --build

8. Xử lý sự cố

LỗiCách xử lý
Port 5055 đã dùngDừng process cũ hoặc đổi BACKEND_PORT
MySQL failedBật MySQL XAMPP, kiểm tra DATABASE_URL
OCR lỗiKiểm tra TESSERACT_CMD và ngôn ngữ vie
Google loginKhớp GOOGLE_REDIRECT_URI với Google Console

1. System requirements

ComponentNotes
Python 3.10+Flask backend
MySQL / MariaDBXAMPP or standalone
Tesseract OCRRequired for image / scanned PDF
Word or LibreOfficeOptional — DOCX to PDF

2. Clone & install dependencies

git clone https://github.com/duyvo26/ai-translation-system.git
cd ai-translation-system/api_base
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

3. Database

Create database ai_translation (utf8mb4). See docs/DATABASE_SETUP.md.

cd api_base
python app/models/base_db.py --check
python app/models/base_db.py

4. Configure .env

cd api_base
copy .env.example .env
  • DATABASE_URL — MySQL connection
  • GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET — Google OAuth
  • OPENAI_API_KEY / DEEPL_API_KEY / GEMINI_API_KEY — AI translation
  • TESSERACT_CMD, OCR_LANGS_DEFAULT=eng+vie — OCR
  • SEPAY_API_KEY, PAYMENT_BANK_* — payments (see docs/SEPAY_INTEGRATION.md)

5. Tesseract (Windows)

  1. Install Tesseract with English + Vietnamese packs
  2. Set TESSERACT_CMD=C:\\Program Files\\Tesseract-OCR\\tesseract.exe

6. Start server

cd api_base
python run_api.py

Default: http://127.0.0.1:5055 (frontend + API on same port).

7. Docker (optional)

docker-compose up --build

8. Troubleshooting

IssueFix
Port 5055 in useStop old process or change BACKEND_PORT
MySQL failedStart MySQL, verify DATABASE_URL
OCR errorsCheck TESSERACT_CMD and vie language
Google loginMatch GOOGLE_REDIRECT_URI in Google Console
